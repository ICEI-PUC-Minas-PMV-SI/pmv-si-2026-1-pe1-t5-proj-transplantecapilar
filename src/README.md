# Instruções de utilização

## Instalação do Site

A aplicação web foi desenvolvida utilizando HTML, CSS e JavaScript, sem necessidade de backend ou banco de dados externo nesta etapa do projeto.

Para executar o sistema localmente, siga os passos abaixo:

1. Faça o download ou clone do repositório do projeto;
2. Abra a pasta do projeto no Visual Studio Code;
3. Instale a extensão “Live Server”;
4. Clique com o botão direito no arquivo `login.html`;
5. Selecione a opção “Open with Live Server”.

O sistema será aberto automaticamente no navegador padrão.

## Funcionalidades disponíveis

- Cadastro de pacientes;
- Cadastro de médicos;
- Login de pacientes e profissionais;
- Painel do paciente;
- Persistência de dados utilizando LocalStorage;
- Checklist diário de acompanhamento;
- Navegação entre páginas do sistema;
- Upload de imagens;
- Registro de dúvidas do paciente.
