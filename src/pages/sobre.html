HairCare Pro

Acompanhamento Pós-Operatório de Transplante Capilar

Sobre o Projeto

O HairCare Pro é uma aplicação web desenvolvida para auxiliar pacientes e profissionais de saúde no acompanhamento pós-operatório de transplante capilar.

A proposta do sistema é facilitar o registro de informações, o envio de fotos, a comunicação com a equipe médica e o acompanhamento da evolução do paciente durante os primeiros dias de recuperação.

Objetivo

O objetivo principal da aplicação é tornar o acompanhamento pós-operatório mais organizado, seguro e acessível, permitindo que paciente e profissional tenham uma visão mais clara do processo de recuperação.

Desenvolvimento Acadêmico

Este projeto foi desenvolvido no curso de Sistemas de Informação da PUC Minas, na disciplina Projeto - Aplicações Web, como parte das etapas de desenvolvimento de uma solução web.

Projeto orientado pelo professor Roberto Felipe Dias Ferreira da Rocha.
